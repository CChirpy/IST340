About this website

This website is currently under construction! To get a glimpse on what's been changed over time, check out my changelog.

This website was created for the course IST 340: Web Design & Development. It contains course materials, such as a web portfolio consisting a series of web pages dedicated to various web techniques, as well as MindTap assignments from the book, New Perspectives on HTML5, CSS3, and JavaScript, by Patrick M. Carey.

Stylistic choices

Each section of the page is colored with a unique and vibrant block of color to better visualize where each section is located and how much space each of them takes up.

When reading my code, my html files are indented with tabs the size of 5 spaces, while my css files are indented with 3 spaces.
